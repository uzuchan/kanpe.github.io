発表やスピーチで使える，自動スクロールするカンペ（テレプロンプター）アプリです．テキストを貼り付けて目標時間をセットすると，スクロール速度が自動で計算されます．

**デモ（ブラウザですぐ使えます）: https://uzuchan.github.io/kanpe.github.io/**

## 特徴

- **目標時間から速度を自動計算** — 「5分で読み切る」のようにタイマーをセットすると，原稿の長さに合わせてスクロール速度を自動調整．
- **読み上げライン** — 画面中央のマーカーと上下のフェードで「いま読む行」に視線を誘導．
- **ミラーモード** — 左右反転表示に対応（プロンプター用ハーフミラーでの利用を想定）．
- **進捗バー** — 原稿全体のどこまで進んだかを画面上部に表示．
- **フォントサイズ・速度の調整** — 本番中でもその場で変更可能．
- **単一HTMLファイル・依存ゼロ** — フレームワークもビルドも不要．ファイルひとつで動きます．
- **スマホ対応** — タッチ操作・セーフエリア（ノッチ）対応．

## 使い方

1. デモURLを開く（またはこのリポジトリの `index.html` をブラウザで開く）
2. 原稿テキストを入力する
3. 目標時間をセットすると速度が自動計算される．手動での速度調整も可能
4. スタートすると原稿がゆっくりスクロールする

## 工夫した点

- **「目標時間 → ピクセル/秒」の速度換算** — 原稿の描画高さと残り時間からスクロール速度を逐次計算し，話す速さが原稿の長さに左右されないようにした．
- **視線移動を最小にするUI設計** — 固定の読み上げライン＋上下フェードで，目を動かさずに読み続けられる画面構成にした．
- **配布のしやすさ** — あえてフレームワークを使わず単一HTMLにまとめ，URLを共有するだけで誰でも使える形にした．
